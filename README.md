## Feature
- Upload File And Read Frame
- TimeLine Dynamic Cache Loading Frame
- JUST LIKE 剪映's TIMELINE

## Tech Stack
- WebAssmeble：C / C++ working in JavaScript for read frames from video
- FFmpeg：library for video editing

## How To Start Using
```
$ npm install
$ npm run serve
```
